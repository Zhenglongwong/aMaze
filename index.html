aMAZE

Instructions

Welcome to aMAZE! The aim of the game is simple! Navigate your way (as an ambitious red square) around a maze to find the yellow objectives hidden around. Collect all the objectives in the shortest amount of time to win! Press Maze 1 or Maze 2 to begin playing!

Controls:

Arrow keys to move, spacebar to start and stop movement.

Create a Maze:

Use the buttons below or their respective keyboard shortcuts to toggle functions and put together the maze of your dreams!

  • Choose Start ('A'): Select the starting point of the player
  • Create Objective ('S'): Click on the board to create an objective
  • Draw ('D'): Hover over the board to draw your maze (use 'D' key to turn this off and on as you draw)
  • Delete ('F'): Click on mistakes to delete them
  • Empty Board: Creates a fresh canvas (erases everything currently on the board)
  • Start Game: Conceals the maze and allows your player to start moving with arrow keys
  • Enter Edit Mode ('G'): Stops an on-going game and reveals the maze, allowing you to make edits. (Works on maze 1 or 2)

Save Your Work:

Use the 'Save Current Maze' button to save your maze for next time. This adds it as a playable maze that you can edit using 'Enter Edit Mode' (just press 'G'). Using the name of an existing maze when saving will override that maze. You can also delete existing mazes.
